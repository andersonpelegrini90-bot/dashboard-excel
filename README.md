Dashboard de Vendas do Xbox com Excel
Projeto desenvolvido como parte do laboratório “Criando um Dashboard de Vendas do Xbox com Excel” da DIO, com foco em organização, análise e visualização de dados de vendas.
​

Objetivo
O objetivo deste projeto é transformar uma base de dados bruta em um dashboard interativo no Excel, permitindo:

Visualizar o desempenho de vendas ao longo do tempo.

Identificar produtos e categorias com melhor performance.

Apoiar a tomada de decisão com indicadores visuais claros.
​

Base de dados
Arquivo original: base.xlsx (fornecido pela DIO).

Principais campos (exemplos, ajuste conforme sua base):

Data da venda

Produto / Categoria

Plataforma (Xbox)

Quantidade vendida

Valor da venda / faturamento

A base foi utilizada como fonte para criação de Tabelas Dinâmicas e gráficos no Excel.
​

Estrutura do Dashboard
O dashboard foi construído no Excel utilizando:

Tabelas Dinâmicas para sumarização dos dados.

Gráficos dinâmicos para visualização de indicadores.

Segmentações de dados (slicers) e filtros para interação.

Formatação e organização da área de Dashboard para melhor visualização.
​

Indicadores que podem estar presentes (ajuste conforme o que você criou):

Faturamento total.

Quantidade total vendida.

Vendas por período (mês/ano).
​

Como reproduzir o projeto
Faça o download do arquivo desafio dio.xlsx.

Abra o arquivo no Microsoft Excel (ou compatível).

Navegue até a aba “Dashboard” para visualizar os gráficos e indicadores.

Utilize os filtros e segmentações para explorar diferentes visões dos dados.

Caso queira reconstruir do zero, utilize também a base original base.xlsx e refaça as Tabelas Dinâmicas e gráficos seguindo a estrutura do dashboard.
